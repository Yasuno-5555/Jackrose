# Cidre v0.25.0: Recovery Screen & Safe Mode Pack

## Summary

v0.25.0 adds a recovery screen, safe mode shell path, recovery actions, and recovery reporting.

## Added Scripts

- `scripts/cidre-panic`
- `scripts/cidre-recovery-screen`
- `scripts/cidre-safe-mode`
- `scripts/cidre-safe-shell`
- `scripts/cidre-session-failure`
- `scripts/cidre-desktop-failure-detect`
- `scripts/cidre-recovery-actions`
- `scripts/cidre-emergency-banner`
- `scripts/cidre-recovery-report`

## Non-goals

- kernel boot failure recovery
- rescue slot implementation
- GUI rescue compositor

## Next

Future work should add a Rescue Slot for failures that cannot reach Cidre scripts at all.
