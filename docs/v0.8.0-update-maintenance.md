# Cidre v0.8.0 - Update & Maintenance Pack

Cidre v0.8.0 is a maintenance release that introduces safe updates, daily maintenance status checks, snapshot pruning, configuration drift detection, and post-update diagnostics.

## Features & Commands

* **cidre-update**: Safer updates for Cidre-managed packages and configurations.
  * `--check`: Verify system update readiness and requirement dependencies.
  * `--dry-run`: Preview changes, package updates, and configuration adjustments.
  * `--apply`: Run pre-update snapshot, sync packages, run configuration apply, and diagnose post-update health.
  * `--doctor`: Runs `cidre-doctor --maintenance` directly.
* **cidre-maintenance**: Consolidates everyday maintenance tasks.
  * `status`: Reports snapshots count, log file sizes, and general state information.
  * `prune`: Safely cleans up stale snapshots and logs. Keeps the latest snapshot protected.
  * `drift`: Tracks modifications between source configurations, deployed configurations, and the installation manifest.
  * `logs`: Inspects execution history of doctor, setup, install, and updates.
* **cidre-doctor --maintenance**: New diagnostic suite verifying state layout, metadata databases, configuration drift, and stale archives.
