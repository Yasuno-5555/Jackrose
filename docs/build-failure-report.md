# Build Failure Report

This document maps out the failure diagnostic details compiled when a builder run encounters errors.

## Collected Metadata

- **Run Context**: Environment variables, host OS, architecture.
- **Trace Output**: Exit code status and the tail end (last 100 lines) of stdout/stderr logs.
- **Debugging Tips**: Path pointers, doctor checks, recommended manual command run scripts.
- **Output Path**: `.local/state/cidre/image-build/build-failure-report.md`
