# macOS Uninstall Guide

This guide is generated from macOS-side restore artifacts.

## Before deleting anything

1. Confirm macOS boots.
2. Confirm your data is backed up.
3. Review Startup Disk.
4. Review advisory partition candidates.
5. Do not touch protected macOS partitions.
6. Keep the Linux-side state export if available.

## Current Limit

Cleanup remains manual in v0.24.0.

If the main Cidre install does not boot, use Rescue Slot first to export state before proceeding with macOS-side cleanup.
