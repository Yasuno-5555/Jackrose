# Cidre v0.24.0: macOS Restore Assistant Pack

## Summary

v0.24.0 adds macOS-side restore readiness checks, partition audit, startup disk guidance, uninstall guide generation, and restore reporting.

## New Scripts

- `scripts/cidre-macos-restore-check`
- `scripts/cidre-macos-partition-audit`
- `scripts/cidre-macos-startup-disk-check`
- `scripts/cidre-macos-uninstall-guide`
- `scripts/cidre-macos-restore-report`
- `scripts/cidre-macos-risk`

## Non-goals

- partition deletion
- APFS resize
- startup disk automatic modification
- destructive guided uninstall

## Next Steps

Use these artifacts as the safety layer for a future guided uninstaller.
