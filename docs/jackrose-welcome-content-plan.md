# Jackrose Welcome Content Plan

## Overview

This plan defines the user guidance and shortcut descriptions integrated into the Jackrose first-login Welcome experience (`jackrose-welcome`). It aligns the documentation and interactive launcher with the active niri baseline configuration.

## Desktop Basics

The welcome dashboard displays a concise list of primary shortcuts on startup:

- **Show shortcuts**: `Mod + Shift + /`
- **Open terminal**: `Mod + Enter` (launches foot)
- **Open launcher**: `Mod + D` (launches fuzzel)
- **Close window**: `Mod + Q`
- **Move focus**: `Mod + H / J / K / L`
- **Open overview**: `Mod + O`

These values must be kept in sync with `/usr/share/jackrose/defaults/niri/config.kdl`.

## Advanced Shortcuts

Advanced options are documented but hidden from the primary list to prevent cognitive overload. Users can discover them via the hotkey overlay:

- **Fallback terminal**: `Mod + Alt + T` (launches foot)
- **Fullscreen**: `Mod + F`
- **Maximize column**: `Mod + Shift + F`
- **Toggle floating**: `Mod + V`
- **Reload niri config**: `Mod + Shift + R`
- **Quit niri**: `Mod + Shift + E`
