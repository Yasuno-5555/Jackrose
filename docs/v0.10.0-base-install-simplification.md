# Cidre v0.10.0 - Base Install Simplification Pack

Cidre v0.10.0 bridges the gap between fresh ALARM minimal installs and the interactive Cidre guided installer by simplifying pre-install system preparations.

## Major Changes

* **Preinstall Setup Wrapper (`preinstall` / `scripts/cidre-preinstall`)**:
  * Run as root from fresh system deployments to check network connectivity, synchronize pacman databases, install base dependencies (`git`, `curl`, `sudo`, `base-devel`), configure wheel sudo privileges, and create user accounts.
* **Installer Warning Redirect**:
  * Root execution attempts on `./install` are intercepted and redirected to run `./preinstall --prepare` first.
* **Base System Diagnostics Mode (`cidre-doctor --base-readiness`)**:
  * New diagnostic mode checking user states, internet reachability, DNS lookups, clock sanity, wheel groups, disk spaces, and Asahi hardware signatures.
