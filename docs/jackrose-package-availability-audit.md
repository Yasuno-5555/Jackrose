# Jackrose Package Availability Audit

## Purpose
This document records the availability status of packages within the Jackrose ecosystem on ALARM / Arch Linux ARM aarch64.

## Buckets
- **Bucket A (Confirmed)**: Available in standard ALARM/Arch repositories and verified to install.
- **Bucket B (Jackrose package required)**: Missing or broken upstream on aarch64, requiring a custom Jackrose PKGBUILD.
- **Bucket C (Deferred)**: Fragile, unverified, or heavy packages postponed for future evaluation.
- **Bucket D (Rejected)**: Outside Jackrose scope or rejected for system integration.
