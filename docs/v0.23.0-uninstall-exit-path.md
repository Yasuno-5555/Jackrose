# Cidre v0.23.0: Uninstall & Exit Path Foundation Pack

## Summary

v0.23.0 adds the Linux-side foundation for safe Cidre removal planning.

## Added

- `scripts/cidre-uninstall-check`
- `scripts/cidre-exit-plan`
- `scripts/cidre-state-export`
- `scripts/cidre-partition-audit`
- `scripts/cidre-macos-restore-guide`
- `scripts/cidre-uninstall-risk`
- `scripts/cidre-goodbye`
- `scripts/cidre-erase-preflight`

## Non-goals

- deleting partitions
- resizing APFS containers
- changing boot policy
- automatic macOS restore

## Validation

The release expects shell syntax checks, export generation, partition audit generation, risk classification, doctor integration, and recovery integration.

## Next

v0.24.0 should deepen the macOS-side restore assistant path.
