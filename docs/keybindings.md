# Cidre Keybindings

This document describes the default keybindings for the Cidre desktop environment. All actions use the `Super` (Windows/Cmd) key as the main modifier.

## Basic Application Launches
- `Super + Enter`: Open Ghostty Terminal (Main)
- `Super + Alt + T`: Open Fallback Terminal (foot)
- `Super + D`: Open fuzzel Application Launcher
- `Super + Q`: Close active window
- `Super + Shift + E`: Logout (Quit graphical session)
- `Super + Shift + R`: Reload Niri Configuration

## Window / Column Layout
- `Super + F`: Toggle Fullscreen window
- `Super + Shift + F`: Maximize column
- `Super + M`: Maximize window to edges (no gaps or borders)
- `Super + Arrow Keys` (or `H`/`J`/`K`/`L`): Focus columns and windows (Left, Down, Up, Right)
- `Super + Ctrl + Arrow Keys` (or `Ctrl+H/J/K/L`): Move columns and windows in layout directions

## Audio & Screen Brightness
- `Volume Up (F12)`: Increase volume by 10% (wpctl)
- `Volume Down (F11)`: Decrease volume by 10%
- `Mute Speaker (F10)`: Toggle audio output mute
- `Mute Mic (F9)`: Toggle microphone input mute
- `Brightness Up (F2)`: Increase screen brightness by 10% (brightnessctl)
- `Brightness Down (F1)`: Decrease screen brightness by 10%

## Screenshots
- `Print`: Take interactive/full screenshot via niri
- `Shift + Print`: Screenshot current active screen
- `Super + Print`: Screenshot current active window
- `Ctrl + Print`: Full screenshot to clipboard (grim | wl-copy)
- `Ctrl + Shift + Print`: Area screenshot selection to clipboard (grim + slurp | wl-copy)
