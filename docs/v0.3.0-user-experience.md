# Cidre v0.3.0: User Experience Pack

Cidre v0.3.0 builds upon the v0.2.0 Bootstrap foundation, focusing on establishing a unified user experience for desktop sessions.

## Key Enhancements

### 1. Unified Visual Aesthetic (Catppuccin RICE)
- Standardized status layout design (Waybar, fuzzel launcher, Ghostty terminal themes) based on the Catppuccin Mocha color scheme.
- Included the default high-fidelity, dark-mode abstract wallpaper (`cidre-wallpaper.png`).

### 2. Standardized Desktop Shortcuts & Fallback
- `Super + T`: Launches Ghostty terminal (utilizes Fish shell by default).
- `Super + Alt + T`: Fallback terminal shortcut mapped to `foot` for quick recovery situations.
- `Super + D`: Launches fuzzel application menu containing desktop applications (Ghostty, Firefox, Thunar, Zed, Helix helper).

### 3. Fcitx5 Mozambique Systemd Integration
- Keyboard layout switches are bound to standard system inputs (`jp106`, Caps Lock mapped to Control).
- Shifted Fcitx5 starting mechanism to a dedicated user systemd service (`fcitx5.service`), making session load times faster and less prone to compositor crash loops.

### 4. Interactive Welcome Dashboard
- Integrates a startup menu CLI (`cidre-welcome`) triggered upon a user's first interactive shell launch to explain standard keyboard shortcuts, input controls, audio configurations, and diagnostic/recovery commands.
