# Upstream niri Notes

Cidre builds on upstream `niri`.

This file exists to keep upstream background material available without making the main Cidre README read like an upstream compositor landing page.

## Why It Matters

Upstream `niri` still provides the baseline compositor model behind `niri-cidre`:

- scrollable tiling
- dynamic workspaces
- monitor-local workspace flow
- core Wayland compositor behavior

If you want deeper background on those foundations, use upstream material.
