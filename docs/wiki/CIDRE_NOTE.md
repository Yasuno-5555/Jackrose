# Documentation note

This documentation includes material inherited from upstream `niri`.

In Jackrose, the standard compositor binary is `niri-jackrose`, and the standard login session is started through `jackrose-session`.

When upstream `niri` documentation refers to `niri`, read it as upstream behavior unless the page explicitly mentions Jackrose.
For Jackrose-specific installation, recovery, package layout, and session behavior, see:
