# Recovery Shortcuts

These are future convenience shortcuts, not the primary recovery path.

## Candidates

- `Super+Shift+Esc`: recovery screen
- `Super+Alt+Enter`: fallback terminal
- `Super+Alt+R`: recovery actions
- `Super+Alt+Backspace`: restart shell/session layer

## Limitation

If the compositor is dead, shortcuts are irrelevant.
TTY and safe mode remain the primary recovery path.
